What?

Slick.js is a CSS Selector Engine. You can use it to find elements in a document using CSS selectors. It supports lots of selectors.

Why?

  • Proven Accuracy (See SlickSpec)
  • No US Sniffing
  • Dynamic per-document Feature Testing
  • Standalone Parser
  • Speed
  • Custom Pseudos
  • Custom Attribute Getters
  • Additional Selector Support
  • Reverse Combinators

SlickSpec

We care so much about the accuracy of css selector parsing and dom search engines that we created a spinoff project called "SlickSpec" to allow any selector engine to collaborate on the same tests and verify that all engines conform to the same specifications and support all the same stuff.

The SlickSpec currently contains over 1,300 specs, each of which contain multiple tests.

These tests covers more edge cases, browser bugs and generally weird stuff than you would ever run across yourself in a lifetime of JavaScript programming.
